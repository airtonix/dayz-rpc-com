# example

A DayZ modification.

## Installation

This mod is managed by the mod scaffolder and is located in `pkgs/example`.

## Development

Edit the mod files in this directory. The mod will be symlinked to the `mods/` directory for server use.

## Structure

- `mod.cpp` - Mod configuration file
- `config/` - Configuration files
- `scripts/` - Server scripts
- `gui/` - GUI resources (imagesets, types)
